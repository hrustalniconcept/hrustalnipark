## Деплой на GitHub Pages

```bash
# 1. Залить файлы в новый репозиторий
git init
git add .
git commit -m "Лендинг квартиры 67 м² — Хрустальный парк, Датский квартал"
git branch -M main
git remote add origin https://github.com/USER/REPO.git
git push -u origin main

# 2. Включить GitHub Pages в настройках репозитория
# Settings → Pages → Source: main / root
# или через gh CLI:
gh api -X POST /repos/USER/REPO/pages -f source.branch=main -f source.path=/

# Сайт будет доступен по адресу:
# https://USER.github.io/REPO/
```

## Что обновлять при изменении цен

При смене акции или базовой цены подменить в двух местах `index.html`:
1. Hero-блок — главная цифра
2. Форма заявки / описание

При обновлении лид-магнита — заменить файл в `downloads/`, имя сохранить (`planirovki-3-scenarii.pdf`), чтобы не сломать ссылки.
