# Планировки · ЖК «Хрустальный парк» — Датский квартал

24 макета планировок A4 (горизонтальные) в стиле сайта hrustalnipark.ru:
Playfair Display + Manrope, акцент `#52358B`, белый фон.

## Структура

```
scripts/
  fetch_fonts.py   — скачивает Playfair Display + Manrope (variable) и нарезает статические веса
  lots.py          — данные всех 24 лотов + группировка по домам
  render.py        — CSS-дизайн-система + HTML-шаблон одного листа
  build.py         — собирает все PDF и 150dpi-превью
fonts/             — нарезанные TTF (Playfair 400/700/900, Manrope 400/500/600/700)
outputs/           — готовые PDF
previews/          — prev_01..prev_24.png для проверки
```

## Сборка

```bash
pip install weasyprint pypdfium2 Pillow openpyxl numpy fonttools requests --break-system-packages
python3 scripts/fetch_fonts.py
python3 scripts/build.py
```

## Дизайн-система

- **Шрифты:** Playfair Display (заголовки, цифры площадей), Manrope (тело, экспликация).
- **Цвета:** `--purple #52358B`, `--ink #2C2630`, `--line #D9CFC0`, `--brass #A8895C`, `--light #8A8290`, фон `#FFFFFF`.
- **Раскладка:** A4 landscape 297×210 мм. Слева (57%) — область под планировку с placeholder. Справа (43%) — данные: вордмарк «Хрустальный парк», субмарка «Датский квартал», номер квартиры, блок площадей (общая = жилая + терраса), chips, экспликация, дисклеймер, футер застройщика.

## Заметки

- **Логотип «Хрустальный парк»** в исходной папке Google Drive отсутствовал (был только логотип девелоперской группы). Вордмарк отрисован живым текстом Playfair Display — совпадает с типографикой ТЗ и стилем сайта, полностью редактируем.
- **Общая площадь = жилая (БТИ) + терраса** без коэффициента. Цен в макетах нет. Дисклеймер: «Цены и точные площади уточняйте у менеджера».
- Застройщик везде: **ООО СЗ «Хрустальный Девелопмент»**.
- Для нескольких лотов (кв. №3 дома 4/14, кв. №4 дома 5/12/13, кв. №4 и №3 дома 6/9, кв. №3 дома 7/8) значение «жилая» по БТИ из ТЗ больше суммы помещений в экспликации — числа оставлены строго как в источнике.
